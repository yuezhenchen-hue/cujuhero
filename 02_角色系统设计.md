## 一、角色体系总览

### 1.1 设计理念

水浒传108将是天然的角色池。我们的设计原则：

1. **忠于原著性格** — 每个角色的蹴鞠能力必须映射其原著特征
2. **位置有意义** — 前锋/中场/后卫/门将不只是标签，直接影响弹射物理和站位
3. **稀有度不等于碾压** — R卡组合得当可以打败SSR，避免纯氪金制胜
4. **羁绊讲故事** — 羁绊组合来自原著关系，让玩家在玩游戏的同时了解水浒故事
5. **攻守皆有操作** — 后卫/GK不是摆件；防守方在对手回合有紧急变阵和手动扑救操作（v1.1）
6. **PVP天梯全标准化** — 天梯赛中所有武将统一Lv.40/★3/无装备，仅保留技能机制差异（v1.1）

### 1.2 基础属性体系

每个武将有 **6项核心属性**（范围 1~100）：

| 属性 | 英文 | 物理引擎映射 | 说明 |
|------|------|-------------|------|
| **力量** | STR (Strength) | 碰撞质量mass、击退距离 | 弹射撞击的力度，击退敌方的能力 |
| **速度** | SPD (Speed) | 弹射最大速度、摩擦系数 | 弹射后移动速度，影响到达球的时间 |
| **技巧** | TEC (Technique) | 弧线球spin强度、连击加成 | 控球能力，弧线球弯曲程度 |
| **防御** | DEF (Defense) | 碰撞体半径、拦截成功率 | 拦截球和抵抗撞击的能力 |
| **意志** | WIL (Willpower) | 技能充能速度、暴击率 | 技能蓄力速度，逆境时触发暴击 |
| **守门** | GK (Goalkeeping) | 守门AI反应速度、扑救范围 | 仅守门员有效的属性 |

**属性与物理引擎的映射公式：**

```python
def calculate_physics_params(hero_data):
    """从角色属性计算物理参数"""
    
    # 质量：力量越高越重（影响碰撞）
    mass = 1.5 + (hero_data.STR / 100) * 3.0      # 范围：1.5 ~ 4.5
    
    # 碰撞半径：防御越高体型越大
    radius = 30 + (hero_data.DEF / 100) * 25        # 范围：30 ~ 55
    
    # 摩擦系数：速度越高摩擦越小（滑得更远）
    friction = 0.99 - (hero_data.SPD / 100) * 0.04  # 范围：0.95 ~ 0.99
    
    # 弹性系数：速度型弹性好
    restitution = 0.7 + (hero_data.SPD / 100) * 0.2 # 范围：0.7 ~ 0.9
    
    # 弧线球强度：技巧映射
    spin_power = hero_data.TEC * 0.02                # 范围：0 ~ 2.0
    
    # 踢球力量加成
    kick_power = 1.0 + (hero_data.STR / 100) * 0.5  # 范围：1.0 ~ 1.5
    
    # 拦截率基础值
    intercept_base = hero_data.DEF / 100 * 0.5       # 范围：0 ~ 0.5
    
    # 暴击率：意志映射
    crit_rate = hero_data.WIL / 100 * 0.25           # 范围：0 ~ 0.25
    
    # 技能充能：意志越高越快
    skill_charge_rate = 1.0 + hero_data.WIL / 100    # 范围：1.0 ~ 2.0
    
    return PhysicsParams(mass, radius, friction, restitution, 
                         spin_power, kick_power, intercept_base, 
                         crit_rate, skill_charge_rate)
```

### 1.3 位置系统

| 位置 | 站位区域 | 弹射特点 | 防守互动（v1.1） | 场上人数 |
|------|---------|---------|------------------|---------|
| **前锋 (FW)** | 对方半场前1/3 | 高速度/力量，适合射门 | — | 1~2 |
| **前腰 (AM)** | 中场偏前 | 高技巧，弧线球/传球 | — | 0~1 |
| **中场 (MF)** | 中场区域 | 均衡，连接前后场 | — | 1~2 |
| **后卫 (DF)** | 我方半场后1/3 | 高防御，拦截/阻挡 | **可紧急变阵+触发式被动** | 1~2 |
| **守门 (GK)** | 球门前 | AI/手动扑救 | **手动扑救（PVP钻石+）** | 1（固定） |

> **v1.1 后卫/GK定位升级：** 后卫不再是"永远不弹射的摆件"。防守回合中，后卫是唯一可被紧急拖拽的位置，
> 且其被动技能改为**触发式**——需要玩家在对手射门时主动点击激活（详见01_物理引擎 5.4）。
> 守门员在高段位PVP中由玩家手动操控扑救方向，增加了控制感和技能表达空间。

**标准阵容：5人上场（1GK + 4外场）**

常用阵型：
```
1-2-1 进攻型：          1-1-2 防守型：          2-1-1 均衡型：
    [FW]                    [FW]                [FW] [FW]
  [AM] [MF]                 [MF]                   [AM]
    [DF]                  [DF] [DF]                [DF]
    [GK]                    [GK]                   [GK]
```

### 1.4 品质/稀有度

| 品质 | 颜色 | 属性总和范围 | 技能数 | 获取难度 | 数量 |
|------|------|------------|--------|---------|------|
| **SSR（天罡星）** | 金色 | 380~450 | 主动+被动+终极 | 极难 | 12将 |
| **SR（地煞星·上）** | 紫色 | 320~380 | 主动+被动 | 较难 | 24将 |
| **R（地煞星·中）** | 蓝色 | 260~320 | 主动 | 普通 | 36将 |
| **N（地煞星·下）** | 绿色 | 200~260 | 被动 | 容易 | 36将 |

> **平衡原则：** N卡虽然数值低，但有些被动技能在特定阵容中无可替代。
> 例：N卡「白日鼠白胜」被动"鼠窃"可以偷取对方buff，反制SSR队伍。

**主动技能 —「天巧花球」（冷却3回合）**
```
效果：本次弹射时，球获得极强旋转
物理修改：
  - ball.spin = 正常值 × 3.5
  - MAGNUS_COEFFICIENT 临时变为 0.4（正常0.15）
  - 球轨迹变为S形双弧线（先左弯再右弯）
  
视觉：球身缠绕金色丝线轨迹，划出优美弧线

**被动技能 —「风流浪子」**
```
效果：燕青每次触球，球速不衰减反而+5%
条件：球经过燕青碰撞体时触发
叠加：最多叠加3次（+15%）
物理修改：ball.friction 临时变为 1.0（不衰减）持续0.5秒
```

**终极技能 —「蹴鞠真意」（需蓄力5回合）**
```
效果：下一次射门无法被扑救
物理修改：
  - 球变为"幻影球"，穿透守门员碰撞体
  - 球身附带5个残影（视觉），实际只有1个真球
  - 守门员AI的predict函数接收假位置，必定扑错方向
  
视觉：球化为五色蹴鞠，分裂成5个分身飞向球门
```

**立绘描述：**
```
外观：俊美青年，发髻上簪花，白衣飘飘
蹴鞠动作：单脚颠球，球在脚尖旋转发光
表情：自信的微笑，眼神灵动
配色：主白色 + 金色点缀 + 淡蓝飘带

N级武将属性总和低（200~260），但每个都有一个 **独特的被动技能**，在特定场合极有价值：

| 代表 | 被动技能 | 战术价值 |
|------|---------|---------|
| 白日鼠白胜 | 「偷天换日」— 30%概率复制对手上一个技能效果 | 反制SSR |
| 段景住（金毛犬） | 「偷马」— 每回合偷取对方10点速度给我方 | 持久消耗 |
| 石勇（石将军） | 「送信」— 教练技能冷却-2 | 加速吴用 |
| 李立（催命判官） | 「毒酒」— 碰到的敌方武将3回合内每回合属性-5 | 持续减益 |
| 曹正（操刀鬼） | 「屠户」— 对低血量（残血）BOSS伤害+100% | PVE专用 |
| 朱贵（旱地忽律） | 「暗号」— 增加我方伏击类技能触发率 | 配合邹渊邹润 |
| 宋万 | 「老资格」— 第一回合全队DEF+15 | 开局抢节奏 |
| 杜迁 | 「摸金校尉」— 赛后金币奖励+50% | 刷金专用 |

> **设计目的：** N卡不是废卡。"白日鼠白胜"在PVP中可以反制氪金玩家的SSR技能，这种设计保证了策略性。

## 七、角色养成系统

### 7.1 成长路线

```
等级提升（Lv.1 → Lv.60）
  │  消耗：经验丹 → 每升1级，6项属性各+0.5~1.0（依品质）
  │
  ├── 星级突破（★1 → ★6）
  │    消耗：同角色碎片
  │    ★1→★2: 10碎片，属性+5%
  │    ★2→★3: 20碎片，属性+10%，解锁皮肤槽
  │    ★3→★4: 40碎片，属性+15%，解锁被动技能升级
  │    ★4→★5: 80碎片，属性+20%，解锁终极技能（仅SSR/SR）
  │    ★5→★6: 150碎片，属性+25%，解锁专属蹴鞠外观
  │
  ├── 技能升级（Lv.1 → Lv.10）
  │    消耗：技能书 + 金币
  │    每级效果提升（冷却-1、伤害+10%、范围+5%等，因技能而异）
  │
  ├── 装备系统
  │    4个装备槽：
  │    [蹴鞠靴] → 主加 STR/SPD
  │    [护 腕]  → 主加 DEF/WIL
  │    [球 衣]  → 主加全属性（小）+ 特效
  │    [腰 带]  → 主加 TEC/WIL
  │
  │    装备品质：白 → 绿 → 蓝 → 紫 → 金 → 红
  │    每件装备有随机副词条（类似原神圣遗物）
  │
  └── 好感度（与"酒馆"系统联动）
       喂食/对话/赠礼 → 解锁角色故事 + 属性微调
       好感度满级：解锁角色专属剧情关卡
```

### 7.2 升级属性成长表（以SSR燕青为例）

| 等级 | STR | SPD | TEC | DEF | WIL | 总计 |
|------|-----|-----|-----|-----|-----|------|
| Lv.1 | 55 | 82 | 98 | 40 | 78 | 353 |
| Lv.20 | 67 | 97 | 115 | 51 | 93 | 423 |
| Lv.40 | 79 | 112 | 132 | 62 | 108 | 493 |
| Lv.60 | 91 | 127 | 149 | 73 | 123 | 563 |
| Lv.60 ★6 | 114 | 159 | 186 | 91 | 154 | 704 |

> 满养成的SSR属性约为初始的2倍，但N卡满养成约350~400，仍无法在纯数值上超过SSR的初始值。
> 差距通过羁绊和被动技能弥补。

### 7.3 PVP标准化规则（v1.1新增）

为解决养成差距导致的"假公平"问题，PVP天梯赛实行完全标准化：

```
天梯赛标准化规则：
  → 所有武将统一为 Lv.40、★3、无装备
  → 技能等级统一为 Lv.5（中等水平）
  → 仅保留角色技能机制差异（穿透/弧线/AOE等）
  → 取消羁绊数值加成，仅保留特殊效果（如视觉标记）

对比示例（燕青在不同模式）：
┌──────────┬────────┬────────┬────────┐
│ 属性     │ 江湖赛  │ 天梯赛  │ 差距    │
│          │ (保留养成)│ (标准化)│        │
├──────────┼────────┼────────┼────────┤
│ ★6 Lv.60 │ 704    │  —     │        │
│ ★3 Lv.40 │ 524    │ 524    │ 0%     │
│ ★1 Lv.40 │ 493    │ 524    │ +6%补偿 │
└──────────┴────────┴────────┴────────┘

设计目标：天梯赛中，一个只有★1的新玩家与★6老玩家
使用同一角色时物理参数完全一致。胜负取决于操作和策略。
```

> **重要**：江湖赛保留养成差异（等级压Lv.40但保留★级和装备），
> 并增加战力匹配度过滤（差距>15%不匹配），满足养成型玩家展示成果的需求。

## 八、反派角色设计（PVE BOSS阵容）

### 8.1 BOSS体系

每个章节的BOSS是 **水浒传中的反派**，他们也有蹴鞠队：

| 章节 | BOSS | 特殊机制 |
|------|------|---------|
| 第一章 | **镇关西·郑屠** | 肉盾型：碰撞体积巨大，需要连击3次才能撞开 |
| 第二章 | **西门庆** | 金钱型：每回合贿赂我方一名武将使其无法行动 |
| 第三章 | **蔡京** | 权谋型：每回合改变球门位置 |
| 第四章 | **童贯** | 军队型：禁军阵列，6人上场（比玩家多1人） |
| 第五章 | **高俅** | 最终BOSS：蹴鞠大师，拥有所有弧线球技能 |
| 隐藏章 | **方腊** | 魔王型：拥有2个球门需要同时防守 |

### 8.2 高俅BOSS战详细设计

```
【高俅 — 蹴鞠太尉】

阶段1（HP 100%~50%）：
  - 高俅使用极致弧线球，球轨迹为S型
  - 每3回合召唤2名禁军侍卫上场（普通敌人）
  - 特殊：高俅的弹射有2次轨迹预测线（玩家不知道他选哪条）

阶段2（HP 50%~20%）：
  - 高俅进入"太尉模式"，场地变为皇宫
  - 球门缩小一半（更难进球）
  - 高俅每回合释放"禁令"：随机禁止一种我方技能类型

阶段3（HP 20%~0%）：
  - 高俅暴走，弹射速度翻倍
  - 场地出现随机障碍（金元宝、太师椅）
  - 特殊：高俅的守门员变为"殿前司都指挥使"（超强GK）
  
通关奖励：解锁高俅为可用角色（SSR）
```

## 二、弹射交互详细设计

### 2.1 选中武将

```
操作：点击场上我方武将
反馈：
  - 被选中武将放大1.2倍 + 发光描边
  - 其他我方武将变暗
  - 显示该武将属性浮窗（简要：名字+位置+技能CD）
  - 若技能可用，武将头顶显示技能图标闪烁
```

### 2.2 拉弹弓

```
操作：按住武将 → 向反方向拖拽
反馈：
  ┌─────────────────────────────┐
  │                              │
  │   · · · · · · →             │ ← 轨迹预测线（水墨虚线）
  │   · ·                       │
  │  ·                          │
  │ · ↗                         │
  │[武将]  ←──── 弹弓橡皮筋     │ ← 从武将到手指的弹性线
  │    ╲                        │
  │     ╲ 手指位置              │
  │                              │
  │  ┌─────────────┐            │
  │  │ ▓▓▓▓▓▓░░░░ │ 60%       │ ← 力度条（圆弧形，在武将脚下）
  │  └─────────────┘            │
  └─────────────────────────────┘

力度条颜色：
  0~30%   绿色（轻踢）
  30~70%  黄色（中等）
  70~100% 红色（全力）→ 武将蓄力动画更剧烈
  
轨迹预测线：
  - 由水墨点组成，越远越淡
  - 遇到边界显示反弹点（小旋涡标记）
  - 线长度与力度成正比
  
技能指示：
  - 如果选中武将技能可用，轨迹线颜色变为技能色
  - 线尾端显示技能效果范围预览
```

### 2.3 弹射释放

```
操作：松开手指
反馈序列：
  0ms    → 武将弹射动画启动 + 弹射音效 + 屏幕轻微震动
  100ms  → 摄像机开始跟随弹射武将（平滑过渡）
  碰撞时 → 碰撞特效 + 碰撞音效 + 短暂慢动作（0.1秒）
  踢球时 → 踢球特效（水墨飞溅）+ 踢球音效 + 慢动作（0.15秒）
  射门时 → 摄像机拉近球门 + 慢动作（0.3秒）+ 守门员扑救
  进球时 → 全屏庆祝特效 + 震屏 + 进球音效 + 慢动作回放
```

### 2.4 进球庆祝

```
进球时序列：
  1. 球入网瞬间：全屏白闪 + 震动
  2. 0.5秒：慢动作回放进球瞬间（可跳过）
  3. 1秒：进球武将的庆祝动画（每个角色独特）
  4. 2秒：比分更新 + combo数显示
  5. 3秒：回归正常

进球文字特效（水墨字体飞入）：
  普通进球：  「进球！」
  2连击进球：「妙传射门！」
  3连击进球：「精彩配合！」
  4连击进球：「天衣无缝！」
  5+连击进球：「替天行道！！！」（全屏金色水墨爆发）
```

## 四、视觉风格规范

### 4.1 色彩体系

```
主色调：
  ┌────────────────────────────────────┐
  │ 水墨黑 #1A1A2E  — 主背景、文字     │
  │ 宣纸白 #F5F0E8  — 次背景、卡面     │
  │ 朱砂红 #C73E3A  — 强调色、进球特效  │
  │ 靛蓝   #2D5F8A  — 功能色、按钮     │
  │ 琥珀金 #D4A843  — SSR品质、稀有     │
  └────────────────────────────────────┘

品质色：
  SSR → 金色渐变 #D4A843 → #F2D675
  SR  → 紫色渐变 #7B3FA0 → #B47ED8  
  R   → 蓝色渐变 #2D5F8A → #5B9BD5
  N   → 绿色渐变 #3A7D44 → #6BBF59

UI色：
  按钮主色   → 朱砂红 #C73E3A
  按钮次色   → 靛蓝 #2D5F8A
  禁用色     → 灰 #8B8B8B
  成功色     → 翡翠绿 #2D8B5F
  警告色     → 琥珀 #D4A843
  危险色     → 深红 #8B1A1A
```

### 4.2 字体

```
标题字体：思源宋体（衬线，古典感）
正文字体：思源黑体（无衬线，可读性）
数字字体：定制像素字体 或 DIN（力量感）
特效字体：毛笔书法字体（进球、技能名等）

字号规范：
  大标题：36pt
  二级标题：28pt
  正文：20pt
  说明文字：16pt
  极小标注：12pt
```

### 4.3 动效原则

```
1. 水墨主题：
   - 页面切换使用水墨晕染过渡（非淡入淡出）
   - 按钮点击产生墨点飞溅微动效
   - 加载动画为毛笔画圈

2. 物理感：
   - 弹射相关操作必须有物理惯性
   - 球的运动带有微微拖影
   - 碰撞产生粒子飞溅

3. 节奏感：
   - 进球时慢动作（子弹时间）
   - Combo连击时画面逐渐加速
   - 关键碰撞瞬间冻结0.1秒（Hit Stop）

4. 60fps：
   - 所有游戏内动画必须60fps
   - UI动画可以30fps
   - 物理引擎固定60fps timestep
```

## 五、音频设计

### 5.1 背景音乐

| 场景 | 风格 | 乐器 |
|------|------|------|
| 主界面 | 悠扬古风 | 古琴 + 竹笛 + 环境音 |
| 比赛中 | 激昂战鼓 | 大鼓 + 唢呐 + 管弦乐 |
| BOSS战 | 热血紧张 | 电吉他 + 大鼓 + 二胡 |
| Roguelike地图 | 江湖冒险 | 琵琶 + 笛 + 轻鼓 |
| 抽卡/酒馆 | 欢快市井 | 三弦 + 快板 + 人声 |

### 5.2 音效列表

| 操作 | 音效描述 |
|------|---------|
| 拉弹弓 | 弓弦拉紧声（力度越大音调越高） |
| 弹射释放 | "嗖"的破空声 |
| 踢球 | 沉闷有力的踢击声 |
| 球碰壁 | 木板撞击回弹声 |
| 球碰门柱 | 金属清脆撞击声 + 颤音 |
| 进球 | 水墨爆裂声 + 欢呼声 + 鼓声 |
| 拦截 | 短促的铁器碰撞声 |
| 技能释放 | 每个角色独立设计（如林冲是虎啸声） |
| Combo x3 | 三连击专属音效 + 人声"好！" |
| Combo x5+ | 大鼓连擂 + 唢呐高音 |
| 扑救 | 扑地声 + 球弹开声 |

## 一、世界观构建

### 1.1 时代背景

```
时间：北宋宣和年间（约1120年）
地点：东京汴梁（今河南开封）为中心，辐射至全国
政治：徽宗沉溺享乐，奸臣当道——蔡京、童贯、高俅、杨戬"四大祸国之臣"
民间：赋税沉重，匪患四起，各路好汉啸聚山林
蹴鞠：朝廷与民间最盛行的运动，上至天子百官，下至贩夫走卒
```

### 1.2 核心设定：蹴鞠的"隐含权力"

在本作世界观中，蹴鞠不仅是体育运动，更是一种**隐性的权力工具**：

```
「蹴鞠令」——高俅掌权后设立的制度：
  · 每年一度"天下蹴鞠大会"，州府必须派队参加
  · 败者罚没家产充军，胜者获朝廷赏赐
  · 高俅借此打压异己——对手球队中暗藏禁军，胜负操控于掌心
  · 民间称之为"鞠刑"——以蹴鞠之名行刑罚之实

「蹴鞠悟道」——燕青发现的秘密：
  · 蹴鞠之术暗合天罡地煞之数，108种踢法对应108星宿
  · 精通者可以"以鞠通武"——蹴鞠的弧线暗合枪法，弹射暗合拳路
  · 高俅正是悟出此理，从泼皮一路踢到太尉之位
  · 燕青天赋异禀，领悟"蹴鞠真意"——以蹴鞠破蹴鞠，以高俅之术反制高俅
```

### 1.3 核心矛盾

```
表层矛盾：梁山好汉 vs 高俅朝廷 → 蹴鞠大会上的正面对决
深层矛盾：江湖道义 vs 庙堂权术 → 蹴鞠是武力还是权谋的工具？
角色矛盾：
  · 燕青——与高俅亦敌亦师（高俅曾点拨燕青蹴鞠）
  · 宋江——不擅蹴鞠却要领导蹴鞠队，靠"义"而非"技"凝聚人心
  · 林冲——被高俅迫害家破人亡，蹴鞠场是他报仇的战场
  · 高俅——蹴鞠天才与权力怪物的统一体，最终BOSS也是"蹴鞠的镜像"
```

## 二、主线剧本大纲

### 2.0 序章「聚义蹴鞠」

**剧情概要：**

```
场景：梁山泊 · 聚义厅
时间：天下蹴鞠大会征召令发出三日后

高俅颁布新一届"天下蹴鞠大会"令：各路势力必须参加，
败者充军，拒不参加者以叛逆论处。梁山泊首当其冲。

宋江召集众好汉商议，众人多主张武力对抗。
唯独燕青提出："高俅以蹴鞠得势，我等便以蹴鞠夺回公道。
在天子面前踢倒高俅，比攻打京城更有说服力。"

鲁智深第一个响应："好！洒家倒要看看，
拳头能护的球门是不是比城门还结实！"

林冲沉默片刻，攥紧拳头："……好。高俅欺我妻离子散，
我便在蹴鞠场上让他跪地求饶。"
```

**教学关设计（3场）：**

| 关卡 | 教学内容 | 对手 | 剧情功能 |
|------|---------|------|---------|
| 序-1「初试身手」 | 弹射基础：拉弓→瞄准→释放 | 训练木桩 | 燕青教玩家基本操作 |
| 序-2「配合之道」 | 连击传球：队友触球combo | 鲁智深（友军对练） | 引出combo和传球概念 |
| 序-3「替天行道」 | 射门得分+技能释放 | 梁山新兵队（简单AI） | 正式组队，出征前的热身赛 |

```
  "哈哈！小燕子，又找洒家练球？
   来来来，让洒家看看你找的新人成色如何！"
```

### 2.2 第二章「州府联赛」

**主题：** 遭遇权谋，认清现实的残酷

**叙事线：**

```
地点：济州 → 青州 → 东平府

梁山蹴鞠队进入正式的州府联赛环节。
对手不再是乡间恶霸，而是由官府支持的正规队伍。

蔡京在幕后操纵裁判和比赛规则，
西门庆用金钱收买梁山队友——
企图从内部瓦解梁山蹴鞠队。

核心剧情：
  · 济州赛：裁判被收买，梁山多次进球被判无效
    → 燕青用超华丽的弧线球让裁判无法否认
  · 青州赛：对手使用"迷香球"（涂了迷药的蹴鞠）
    → 安道全（R级医者）加入，解毒
  · 东平府赛：西门庆试图收买时迁
    → 时迁假意投诚，反过来偷了西门庆的阴谋证据

BOSS1·西门庆：金钱攻势，每回合贿赂一名武将
BOSS2·蒋门神：纯暴力对决，碰撞力×3

解锁Roguelike模式——"江湖历练"开放
叙事理由：吴用说"正式大会前还有时间，
众好汉各自外出历练，磨砺球技。"
```

**第二章关键台词：**

```
———— 济州赛 · 裁判改判 ————

裁判（吹哨）：
  "犯规！梁山队撞人！进球无效！"

鲁智深（怒）：
  "放屁！洒家碰都没碰到他！"

吴用（按住鲁智深，低声）：
  "忍。这个裁判是蔡京的人。
   动手就中了他们的计——以暴行取消参赛资格。"

燕青（闭目，深呼吸，然后睁开眼）：
  "……不需要裁判认可。"
  "我踢一个球，让所有人都认可。"

（燕青释放「天巧花球」——极致弧线球，
  球在半空中划出S型曲线，绕过三名防守者，
  从球门死角飞入。全场观众起立鼓掌。）

裁判（额头冒汗，张嘴想说犯规，
      但全场观众已经在欢呼）：
  "……进球有效。"

燕青（淡淡微笑）：
  "多谢大人——明察秋毫。"

———— 西门庆试图收买时迁 ————

西门庆（密室，推出一箱金子）：
  "时迁兄弟，你我都是聪明人。
   梁山成不了事，何必陪他们送死？
   这些金子，换你一场比赛的情报——如何？"

时迁（眼睛发亮，双手颤抖地去拿金子）：
  "西门大官人出手果然阔绰……"

（突然，时迁手一翻，金子消失，
  转而从西门庆袖中抽出一封信）

时迁（嬉笑）：
  "金子嘛，回头再算。
   但这封信——蔡京给你的密令——
   我先借走了？"

西门庆（大惊）：
  "你——！来人！给我拿下他！"

（时迁已翻窗消失，只留下笑声）

时迁（远处传来声音）：
  "西门大官人，鼓上蚤偷东西，
   天底下还没人追得上！哈哈哈！"
```

### 2.3 第三章「京城淘汰赛」

**主题：** 进入权力中心，面对最强对手

**叙事线：**

```
地点：东京汴梁 · 蹴鞠大会主场

梁山蹴鞠队终于杀入京城。
蹴鞠大会的淘汰赛阶段在皇宫外的大校场举行，天子偶尔莅临。

对手升级为禁军精英、太学名队、达官显贵的私人球队。
蔡京和童贯联手设下层层障碍：

  · 蔡京出招：临时改变比赛规则（球门位置每回合变化）
  · 童贯出招：禁军阵列队（6人上场，比玩家多1人）
  · 高俅暗手：派人散播谣言——"梁山是反贼"，
    煽动京城百姓敌视梁山队

转折：在与禁军队的比赛中，林冲认出了曾经的同僚——
他们被迫为高俅踢球。林冲的愤怒达到顶点。

卢俊义在此章加入：他是京城首富，
因高俅觊觎其家产被诬陷入狱，公孙胜道法救出。
卢俊义倾尽家财支持梁山蹴鞠队。
```

**第三章关键台词：**

```
———— 林冲遇旧友 ————

林冲（震惊）：

王进（低头，不敢对视）：
  "林冲……你不该来的。"

林冲（声音颤抖）：
  "当年你教我枪法，教我做人。
   如今你却在高俅麾下……为虎作伥？"

王进（苦涩）：
  "你以为我愿意？
   我的家人……都在高俅手里。
   我不踢，他们就没命。"

林冲（沉默许久，然后抬头）：
  "……我懂了。那今天这场球——
   我不会手下留情。
   不是为了赢你，是为了救你。"

王进（轻声）：
  "……多谢。"

———— 卢俊义加入 ————

卢俊义（身穿囚衣，被公孙胜救出，站在梁山众人面前）：
  "我卢俊义，河北第一棍。
   半生富贵，一朝入狱，方知世道不公。"
  （接过燕青递来的蹴鞠，掂了掂）
  "蹴鞠么？我练过——河北三绝之一。"
  （一脚将球踢出，球带着火焰般的轨迹飞出）
  "高俅夺我家产，我便在蹴鞠场上——
   连本带利地讨回来。"
```

### 2.4 第四章「皇家决赛」

**主题：** 最终对决，正邪之战

**叙事线：**

```

```

## 五、UI文本规范

### 5.1 文案风格指南

```
总体风格：半文半白——古韵但不晦涩

对话文案：
  ✅ "兄弟，这一脚——有意思。"
  ✅ "高俅此人，蹴鞠是真的强。人品嘛……呵。"
  ❌ "此乃吾等之宿命也！"（太文言）
  ❌ "这波操作666！"（太现代）

  ✅ 按钮文本用四字短语——"出征！""开球！""再来一局""收入囊中"
```

## 七、本地化预留

### 7.1 多语言策略

| 语言 | 优先级 | 文案策略 |
|------|--------|---------|
| 简体中文 | P0（首发） | 半文半白原生风格 |
| 繁体中文 | P0（港台首发） | 同简体风格，繁体转换+用词调整 |

| 韩语 | P2（第三批） | 参考韩版水浒传翻译 |

```
  · 蹴鞠：Cuju（全球统一不翻译）
```

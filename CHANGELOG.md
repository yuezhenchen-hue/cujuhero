## v1.1.0 — 2026-04-10 | 融合竞技玩家评审

### 核心变更

融合 `08_竞技玩家评审报告.md` 提出的6大竞技体验问题，全面重构PVP系统。

### 01_物理引擎设计.md

- **新增 分级轨迹预测线系统**
  - `PredictionLevel` 枚举：FULL / MEDIUM / DIRECTION_ONLY / NONE
  - 按游戏模式+PVP段位动态选择预测精度
  - PVE保持完整预测线（新手友好）
  - PVP大师段位以上无预测线（操作区分度）
- **新增 防守回应系统 `DefenseResponseSystem`（5.4节）**
  - 对手弹射后1秒内可拖拽一名后卫移动（每场限4次）
  - 后卫被动技能改为触发式（需玩家主动点击激活）
  - 消灭"对手回合50%时间看戏"问题
- **新增 手动扑救系统 `ManualGoalkeeperSave`（5.3.2节）**
  - PVP钻石段位以上：球飞向球门时玩家左/右滑动控制守门员
  - 猜对方向扑救率60%~95%（受GK属性加成）
  - 猜错方向几乎必进，增加博弈深度
- **改进 守门员AI弧线球预判**
  - `predict_ball_position()` 加入马格努斯效应偏移补偿
  - GK属性决定预判精度（0~100%映射）
- **新增 反消极比赛系统 `AntiPassiveSystem`（5.5节）**
  - 射门计时器：连续3回合无射门→对方获任意球
  - 球权递进：连续3回合球未被触→球前移200单位
  - 逆风buff：落后2分SPD+10%，落后3分对方球门缩小20%
- **重构 回合管理器为半场制（6节）**
  - 上半场5回合（A先手）→ 中场休息30秒 → 下半场5回合（B先手）
  - 中场可调整阵容/阵型
  - 回合流程中集成防守回应窗口和手动扑救判定

### 02_角色系统设计.md

- **新增 设计原则** #5攻守皆有操作 和 #6 PVP天梯全标准化
- **位置系统增加"防守互动"列**：DF可紧急变阵+触发式被动，GK可手动扑救
- **鲁智深「铁壁禅功」改为触发式被动**：PVP中需玩家手动激活，增加时机判断
- **新增 PVP标准化规则（7.3节）**
  - 天梯赛：Lv.40 / ★3 / 无装备 / 技能Lv.5 统一标准
  - 江湖赛：保留养成，增加战力匹配度（差>15%不匹配）

### 03_关卡与Roguelike系统.md

- **PVP系统全面重构（第三节）**
  - 段位系统增加机制对照表（预测线/扑救/BP等按段位开启）
  - **新增 双PVP模式**：天梯赛（全标准化）+ 江湖赛（保留养成）
  - **新增 完整BP选将系统**：大师段位以上各ban 3 + 交替pick 5
  - **新增 半场制赛制**：上下半场各5回合，先手互换
  - **新增 反消极机制**：射门计时器 + 球权递进 + 逆风buff
  - **新增 combo进球得分**：普通1分 / 3连击2分 / 5连击3分
  - **新增 竞技基础设施**：回放系统 / 观战系统 / 训练场

### 04_UI与UX交互设计.md

- **核心交互流程增加防守方视角**：紧急变阵→手动扑救→回合结算
- **HUD更新**：半场制显示 + 射门计时器
- **新增 防守回应交互UI（2.5节）**：后卫高亮拖拽 + 触发式技能按钮
- **新增 手动扑救交互UI（2.6节）**：三区域滑动操作 + 0.8秒窗口
- **新增 中场休息界面（2.7节）**：上半场数据 + 阵容调整
- **新增 竞技基础设施UI（第六节）**：回放/观战/训练场/BP选将界面
- **音效表新增**：手动扑救提示、紧急变阵提示、半场哨声

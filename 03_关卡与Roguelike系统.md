## 一、PVE主线剧情

### 1.1 世界观

> 北宋末年，奸臣高俅以蹴鞠一技之长获得天子宠信，官至太尉。
> 为巩固权势，高俅设立"天下蹴鞠大会"，名为比试蹴鞠，
> 实则以蹴鞠为名行打压异己之实——凡大会中败北者，罚没家产，流放充军。
> 梁山众好汉不堪压迫，宋江决意组建梁山蹴鞠队，
> 由燕青领衔，一路从州府踢到京城，以蹴鞠之道，行替天行道之事。

## 二、Roguelike模式「江湖历练」

### 2.1 模式概述

通关第二章后解锁。独立于主线，每次进入随机生成路线、敌人、奖励。

- **每局约 15~25 分钟**
- **每局选 3~5 名武将** 进入
- **死亡（被淘汰）重新开始**，但角色养成保留
- **核心循环：比赛 → 获得江湖秘技 → 变强 → 更难的比赛 → ...**

### 2.2 地图结构（类杀戮尖塔）

```
                    [💀 BOSS]
                   ╱         ╲
              [⚽比赛]    [⚽比赛]
             ╱    ╲      ╱    ╲
        [🎁宝箱] [🏥休息] [⚽比赛] [❓事件]
           ╱  ╲      ╲      ╱    ╲
      [⚽比赛] [❓事件] [⚽比赛] [🏪商店]
         ╱    ╲         ╱         ╲
    [🎁宝箱] [⚽比赛] [🏥休息]  [⚽比赛]
              ╲      ╱
               [起点]

节点类型：
⚽ 比赛节点 — 打一场蹴鞠赛，胜利获得秘技
🎁 宝箱节点 — 直接获得秘技/装备
🏥 休息节点 — 恢复队伍状态/升级秘技
🏪 商店节点 — 花金币购买秘技/一次性道具
❓ 随机事件 — 随机剧情，可能获益可能受损
💀 BOSS节点 — 章节BOSS，通过后进入下一层
```

**层数结构：**
```
第1层（乡村）：4~5个节点 + 1 BOSS → 简单
第2层（城镇）：5~6个节点 + 1 BOSS → 中等
第3层（州府）：6~7个节点 + 1 BOSS → 困难
第4层（京城）：4个节点 + 最终BOSS → 地狱
```

### 2.3 江湖秘技（Roguelike Build核心）

每次战斗胜利，从3张秘技中选1张。秘技分为 **攻/守/辅/特** 四类。

| 「弧月斩」 | 稀有 | 弧线球弯曲度+50% | 弧线 |
| 「万箭齐发」 | 史诗 | 射门时球分裂为3个 | 终极火力 |

### 2.4 秘技Build路线设计

**鼓励玩家形成build方向，而非什么都拿一点：**

```
连击流 Build：
  连珠炮 + 梁山义气 + 及时雨 + 万箭齐发
  → 追求极致Combo，一次弹射连击5+次后射门

穿透流 Build：
  穿云箭 + 泰山压顶 + 力拔山 + 流星赶月
  → 无视防守，暴力穿透直接射门

弧线流 Build：
  弧月斩 + 燕青为核心 + 缩地成寸
  → 极致弧线绕过所有障碍

防守反击 Build：
  铁壁 + 金钟罩 + 水泊之盾 + 万夫莫开
  → 防住对手然后反击
```

### 2.5 随机事件（❓节点）

```
「路遇好汉」
  → 一名随机武将想加入你的队伍（可替换现有成员）
  → 或者挑战你（小型1v1比赛），赢了获得稀有秘技

「酒馆小聚」
  → 全队恢复状态
  → 但有30%概率有人喝醉（下一场比赛该武将随机行动）

「黑店劫道」
  → 孙二娘的黑店！
  → 选择：交出一个秘技换取2个新秘技 / 拒绝并战斗

「高俅密探」
  → 发现高俅的间谍！
  → 下一场比赛对手知道你的阵容（显示你的武将属性给AI）
  → 但你可以获得一个"反间计"秘技

「落魄英雄」
  → 遇到一名受伤的好汉（杨志/林冲/武松等剧情角色）
  → 帮助他：消耗金币，但该武将临时加入且全属性+10
  → 忽略他：无事发生

「蹴鞠赌局」
  → 赌你下一场的表现
  → 如果下一场combo达到5+：获得传说秘技
  → 如果没达到：失去一个秘技
```

## 三、PVP竞技系统（v1.1全面重构）

### 3.1 段位系统

```
青铜：蹴鞠学徒（初始段位）
白银：街头蹴客
黄金：州府名将
钻石：京城高手      ← 从此段位起：手动扑救激活 + 预测线缩短
大师：天下蹴王      ← 预测线仅显示方向箭头
传说：替天行道（赛季前100名）← 无预测线
荣耀：蹴鞠圣手（赛季前10名）← 无预测线 + 全部手动扑救
```

**段位与竞技机制对照表：**

| 段位 | 预测线 | 守门员 | 防守回应 | 完整BP |
|------|--------|--------|---------|--------|
| 青铜~黄金 | 完整轨迹+2反弹 | 纯AI扑救 | 不开启 | 简化(各ban 1) |
| 钻石 | 50%长度+1反弹 | AI+手动加成 | 开启 | 标准BP |
| 大师 | 仅方向箭头 | 纯手动扑救 | 开启 | 标准BP |
| 传说/荣耀 | 无 | 纯手动扑救 | 开启 | 标准BP |

### 3.2 双PVP模式

为兼顾硬核竞技玩家和养成型玩家，设两种独立的PVP模式：

#### 3.2.1 天梯赛（标准竞技）

```
定位：纯粹公平竞技，操作和策略决定一切

数值标准化规则：
  → 所有武将统一为 Lv.40、★3、无装备
  → 只保留角色技能机制差异，取消数值碾压
  → 胜负纯粹取决于：阵容搭配 + 弹射操作 + 临场反应

赛制：半场制（上下半场各5回合）
  上半场：A方先手（奇数回合），B方先手（偶数回合）
  中场休息：30秒，可调整阵容/阵型
  下半场：B方先手（先手互换）
  比分合计判定胜负
  平局：加时赛（各1次弹射，金球制）

段位影响：独立天梯分数，赛季奖励更丰厚
```

#### 3.2.2 江湖赛（养成竞技）

```
定位：展示养成成果，允许数值差异

数值规则：
  → 等级压缩至Lv.40（取消等级碾压）
  → 保留★级突破和装备效果
  → 增加「装备匹配度」：双方总战力差>15%不匹配

赛制：同天梯赛半场制
段位影响：独立段位，与天梯赛分开计算
奖励：养成资源（金币、碎片、装备箱）
```

### 3.3 Ban/Pick选将系统

v1.0的"各ban 2名"对108个角色来说影响微乎其微。v1.1升级为完整BP：

```
┌─────────── 完整BP流程（大师段位以上）───────────┐
│                                                  │
│  Ban阶段1：A ban 2 → B ban 2                    │
│  Pick阶段1：A pick 1 → B pick 2 → A pick 1     │
│  Ban阶段2：A ban 1 → B ban 1                    │
│  Pick阶段2：B pick 1 → A pick 2 → B pick 1     │
│                                                  │
│  总计：各ban 3名 + pick 5名 = 8个选择位          │
│  让版本强势角色不会垄断每场比赛                   │
│  冷门角色在特定ban位下有出场价值                  │
└──────────────────────────────────────────────────┘

简化BP（钻石以下）：
  → 各ban 1名 → 同时选5名 → 30秒阵容调整
```

### 3.4 PVP平衡机制

#### 反消极比赛（铁桶阵惩罚）

```
射门计时器：
  → 每方必须在连续3次弹射内完成至少1次射门尝试
  → 否则被判"消极比赛"，对方获得一次任意球
  → 任意球：中场直射球门，仅面对守门员

球权递进：
  → 连续3回合球未被任何武将触及 → 球自动前移到中场
  → 防止双方都不碰球的僵持局面
```

#### 逆风buff（追分机制）

```
落后2分：落后方全队SPD+10%
落后3分：落后方SPD+15% + 领先方球门宽度缩小20%

combo进球得分（鼓励高风险高回报进攻）：
  普通射门 = 1分
  3连击射门 = 2分
  5连击射门 = 3分
  → 落后方可以搏combo高分翻盘
  → 领先方如果只防守不进攻，分差不安全
```

#### 其他平衡规则

```
技能冷却：PVP中所有技能冷却+1回合（鼓励基础操作）
阵容公开：BP结束后双方互看阵容，有30秒调整阵型时间
防守回应：对手弹射后1秒内可拖拽后卫移动（每场限4次）
手动扑救：钻石以上段位开启（详见01_物理引擎 5.3.2）
```

### 3.5 竞技基础设施

#### 回放系统

```
存储：记录每次弹射的输入参数（英雄ID、方向向量、力度）
      + 防守回应操作 + 手动扑救方向
      确定性物理引擎可完美重现整场比赛
回放功能：
  - 慢动作 / 2x加速
  - 自由摄像机角度
  - 逐回合跳转
  - 一键导出GIF/视频（带游戏水印，用于社交传播）
自动保存：最近20场对局自动保存
手动收藏：无上限
```

#### 观战系统

```
好友观战：实时观看好友对局（延迟30秒防作弊）
排行榜直播：赛季前10名对局可公开观战
观战UI：
  - 双方视角切换
  - 弹幕/表情反应
  - 观战人数显示
```

#### 训练场

```
模式：
  - 自由练习：任意放置武将和球，无限弹射
  - 射门挑战：指定位置/角度进球，计时排名
  - 扑救练习：AI射门，练习手动扑救反应
  
特殊功能：
  - 弹射后显示完整轨迹回放（用于学习反弹角度）
  - 不消耗体力
  - 可使用任何已解锁角色（包含未升级的）
  → 这对"高段位无预测线"的适应训练至关重要
```

## 一、PVE主线剧情

### 1.1 世界观

> 北宋末年，奸臣高俅以蹴鞠一技之长获得天子宠信，官至太尉。
> 为巩固权势，高俅设立"天下蹴鞠大会"，名为比试蹴鞠，
> 实则以蹴鞠为名行打压异己之实——凡大会中败北者，罚没家产，流放充军。
> 梁山众好汉不堪压迫，宋江决意组建梁山蹴鞠队，
> 由燕青领衔，一路从州府踢到京城，以蹴鞠之道，行替天行道之事。

### 1.2 章节结构

```
序章「聚义蹴鞠」 — 教学关（3场）
  │  解锁：燕青、林冲、鲁智深
  │  剧情：燕青提议以蹴鞠对抗高俅，众好汉响应
  │
第一章「江湖初赛」（8场 + 1 BOSS）
  │  地点：梁山泊周边乡镇
  │  对手：地方恶霸、土匪球队
  │  BOSS：镇关西·郑屠（肉盾型BOSS）
  │  解锁武将：武松、李逵、花荣
  │  奖励：基础装备、金币、R碎片
  │
第二章「州府联赛」（12场 + 2 BOSS）
  │  地点：济州、青州、东平府
  │  对手：州府官兵队、当地豪绅队
  │  BOSS1：西门庆（金钱型，每回合贿赂一名武将）
  │  BOSS2：蒋门神（力量型，碰撞力×3）
  │  解锁武将：戴宗、杨志、张清
  │  引入：Roguelike模式解锁
  │
第三章「京城淘汰赛」（15场 + 2 BOSS）
  │  地点：东京汴梁
  │  对手：禁军蹴鞠队、太学队、达官球队
  │  BOSS1：蔡京（权谋型，改变球门位置）
  │  BOSS2：童贯（军阵型，6人上场）
  │  解锁武将：公孙胜、卢俊义
  │
第四章「皇家决赛」（8场 + 终极BOSS）
  │  地点：大内皇宫蹴鞠场
  │  对手：殿前司精英队、高俅亲卫队
  │  终极BOSS：高俅（三阶段战斗）
  │  通关奖励：高俅可招募、SSR自选券
  │
隐藏章节（通关后解锁）
  ├── 「方腊远征」— 南方BOSS关卡
  ├── 「辽国邀请赛」— 异域风格对手
  └── 「水浒全明星」— 108将VS108将
```

### 1.4 主线关卡胜利条件类型

| 条件类型 | 说明 | 出现时机 |
|---------|------|---------|
| **进球数** | 在N回合内进M球 | 全程 |
| **比分胜利** | N回合内比分领先 | 全程 |
| **零封** | 不让对方进球且自己进2球以上 | 第二章起 |
| **Combo** | 达成N次以上连击后射门得分 | 第二章起 |
| **BOSS击败** | 使BOSS HP归零（撞击BOSS=伤害） | BOSS关 |
| **限时** | 在30秒内完成进球 | 挑战关 |

## 二、Roguelike模式「江湖历练」

### 2.1 模式概述

通关第二章后解锁。独立于主线，每次进入随机生成路线、敌人、奖励。

- **每局约 15~25 分钟**
- **每局选 3~5 名武将** 进入
- **死亡（被淘汰）重新开始**，但角色养成保留
- **核心循环：比赛 → 获得江湖秘技 → 变强 → 更难的比赛 → ...**

### 2.2 地图结构（类杀戮尖塔）

```
                    [💀 BOSS]
                   ╱         ╲
              [⚽比赛]    [⚽比赛]
             ╱    ╲      ╱    ╲
        [🎁宝箱] [🏥休息] [⚽比赛] [❓事件]
           ╱  ╲      ╲      ╱    ╲
      [⚽比赛] [❓事件] [⚽比赛] [🏪商店]
         ╱    ╲         ╱         ╲
    [🎁宝箱] [⚽比赛] [🏥休息]  [⚽比赛]
              ╲      ╱
               [起点]

节点类型：
⚽ 比赛节点 — 打一场蹴鞠赛，胜利获得秘技
🎁 宝箱节点 — 直接获得秘技/装备
🏥 休息节点 — 恢复队伍状态/升级秘技
🏪 商店节点 — 花金币购买秘技/一次性道具
❓ 随机事件 — 随机剧情，可能获益可能受损
💀 BOSS节点 — 章节BOSS，通过后进入下一层
```

**层数结构：**
```
第1层（乡村）：4~5个节点 + 1 BOSS → 简单
第2层（城镇）：5~6个节点 + 1 BOSS → 中等
第3层（州府）：6~7个节点 + 1 BOSS → 困难
第4层（京城）：4个节点 + 最终BOSS → 地狱
```

### 2.3 江湖秘技（Roguelike Build核心）

每次战斗胜利，从3张秘技中选1张。秘技分为 **攻/守/辅/特** 四类。

#### 攻击类秘技

| 秘技名 | 品质 | 效果 | Build方向 |
|--------|------|------|----------|
| 「烈火踢」 | 普通 | 射门球速+15% | 火力 |
| 「连珠炮」 | 普通 | Combo加成+5%/次 | 连击 |
| 「穿云箭」 | 稀有 | 球穿透第一个防守者 | 穿透 |
| 「力拔山」 | 稀有 | STR+20（全队） | 力量 |
| 「弧月斩」 | 稀有 | 弧线球弯曲度+50% | 弧线 |
| 「万箭齐发」 | 史诗 | 射门时球分裂为3个 | 终极火力 |
| 「泰山压顶」 | 史诗 | 碰撞击退力×2 | 暴力 |
| 「流星赶月」 | 传说 | 弹射后武将变为不灭流星，反弹5次才停 | 超级弹射 |

#### 防守类秘技

| 秘技名 | 品质 | 效果 |
|--------|------|------|
| 「铁壁」 | 普通 | 全队DEF+10 |
| 「铜墙」 | 普通 | 拦截成功率+10% |
| 「金钟罩」 | 稀有 | 每回合第一次被射门无效 |
| 「水泊之盾」 | 稀有 | 球门前产生减速区域 |
| 「龟甲阵」 | 史诗 | 后卫碰撞体积+80% |
| 「万夫莫开」 | 传说 | 对方连续3次射门内必扑出2次 |

#### 辅助类秘技

| 秘技名 | 品质 | 效果 |
|--------|------|------|
| 「梁山义气」 | 普通 | 3名同队好汉上场，全属性+5 |
| 「及时雨」 | 普通 | 技能冷却-1 |
| 「锦囊」 | 稀有 | 每层开始随机获得一个一次性buff |
| 「招贤纳士」 | 稀有 | 可临时招募一名不在队中的武将 |
| 「酒神」 | 史诗 | 武松、鲁智深、李逵全属性+20 |
| 「替天行道」 | 传说 | 全队武将视为梁山好汉（激活所有羁绊） |

#### 特殊类秘技（改变规则）

| 秘技名 | 品质 | 效果 |
|--------|------|------|
| 「乾坤颠倒」 | 稀有 | 上下球门互换（需要新策略） |
| 「双球混战」 | 稀有 | 场上同时出现2个球 |
| 「缩地成寸」 | 史诗 | 场地缩小50%（更紧凑更刺激） |
| 「时光倒流」 | 史诗 | 被进球后有50%概率取消该进球 |
| 「混世魔王」 | 传说 | 每回合随机触发一个全局效果 |

### 2.4 秘技Build路线设计

**鼓励玩家形成build方向，而非什么都拿一点：**

```
连击流 Build：
  连珠炮 + 梁山义气 + 及时雨 + 万箭齐发
  → 追求极致Combo，一次弹射连击5+次后射门

穿透流 Build：
  穿云箭 + 泰山压顶 + 力拔山 + 流星赶月
  → 无视防守，暴力穿透直接射门

弧线流 Build：
  弧月斩 + 燕青为核心 + 缩地成寸
  → 极致弧线绕过所有障碍

防守反击 Build：
  铁壁 + 金钟罩 + 水泊之盾 + 万夫莫开
  → 防住对手然后反击
```

### 2.5 随机事件（❓节点）

```
「路遇好汉」
  → 一名随机武将想加入你的队伍（可替换现有成员）
  → 或者挑战你（小型1v1比赛），赢了获得稀有秘技

「酒馆小聚」
  → 全队恢复状态
  → 但有30%概率有人喝醉（下一场比赛该武将随机行动）

「黑店劫道」
  → 孙二娘的黑店！
  → 选择：交出一个秘技换取2个新秘技 / 拒绝并战斗

「高俅密探」
  → 发现高俅的间谍！
  → 下一场比赛对手知道你的阵容（显示你的武将属性给AI）
  → 但你可以获得一个"反间计"秘技

「落魄英雄」
  → 遇到一名受伤的好汉（杨志/林冲/武松等剧情角色）
  → 帮助他：消耗金币，但该武将临时加入且全属性+10
  → 忽略他：无事发生

「蹴鞠赌局」
  → 赌你下一场的表现
  → 如果下一场combo达到5+：获得传说秘技
  → 如果没达到：失去一个秘技
```

## 三、PVP竞技系统

### 3.1 段位系统

```
青铜：蹴鞠学徒（初始段位）
白银：街头蹴客
黄金：州府名将
钻石：京城高手
大师：天下蹴王
传说：替天行道（赛季前100名）
荣耀：蹴鞠圣手（赛季前10名）
```

### 3.2 PVP比赛规则

```
赛制：3分钟回合制（双方各约10次弹射机会）
上场：5人阵容
先手：随机
回合：双方交替弹射（我方弹射→对方弹射→...）
胜负：时间结束比分高者胜
平局：进入加时赛（各1次弹射，金球制）
```

### 3.3 PVP平衡策略

```
1. 武将等级压缩：PVP中所有武将等级统一为Lv.40
   → 减少数值碾压，突出策略

2. 禁选系统：双方各ban 2名武将
   → 增加策略深度

3. 阵容公开：比赛开始前双方互看阵容，有30秒调整时间
   → 减少"猜拳"成分

4. 技能冷却统一：PVP中所有技能冷却+1回合
   → 防止技能乱飞，鼓励基础操作
```

## 四、赛季与活动系统

### 4.1 赛季周期

```
每赛季 45天，与现实足球赛事联动：

2026年夏季赛季：「世界杯热血季」
  - 限定皮肤：武将穿各国球衣风格服装
  - 限定模式：11v11全员上场的大乱斗
  - 限定武将：「蹴鞠使者」—— 来自丝绸之路的异域球员

2026年秋季赛季：「中秋蹴鞠节」
  - 限定场景：月下蹴鞠
  - 蹴鞠变为月饼外观
  - 限定秘技：月系技能

2027年春节赛季：「水浒闹元宵」
  - 限定场景：元宵灯会
  - 特殊模式：烟花射门（进球触发烟花）
```

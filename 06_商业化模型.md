> 版本：v1.0 | 最后更新：2026-04-10

## 一、商业模式定位

### 1.1 模式选择：Free to Play + 中度付费

```
定位：免费下载 + 内购（无广告打断游戏体验）
目标ARPU：$3~5/月（中等水平，不做重氪）
目标付费率：5~8%
LTV目标：$15~25（6个月）
```

**为什么不做重氪？**
1. 水浒IP的目标用户是文化爱好者，对氪金容忍度较低
2. Roguelike+弹射玩法的核心是操作+策略，Pay to Win会毁掉核心体验
3. 中度付费+高留存+长周期 > 重氪+低留存+短命

## 二、收入来源结构

### 2.1 收入占比目标


### 2.2 内购商品详细设计

#### 💎 货币体系

```
金币（软通货）：
  获取：比赛奖励、每日任务、卖装备
  用途：角色升级、装备强化、商店购买
  
钻石（硬通货）：
  获取：充值、成就、首通关卡、PVP赛季奖励
  用途：抽卡、刷新体力、购买限定道具
  
酒令（抽卡专用）：
  获取：充值赠送、活动获取
  用途：仅用于抽卡
  
充值比例：
  6元   → 60钻石
  30元  → 330钻石（+10%赠送）
  68元  → 780钻石（+15%赠送）
  128元 → 1580钻石（+23%赠送）
  328元 → 4280钻石（+30%赠送）
  648元 → 8980钻石（+38%赠送）
```

#### 🍶 招募系统（抽卡）

```
单抽：160钻石 / 1张酒令
十连：1440钻石 / 10张酒令（9折）

概率：
  SSR：2.0%（UP角色占50% = 1.0%）
  SR ：10.0%（UP角色占50% = 5.0%）
  R  ：58.0%
  N  ：30.0%
  
保底机制：
  小保底：10连必出SR以上（如果10连没有紫色以上，第10发必出SR）
  大保底：80抽必出SSR
  UP保底：如果大保底歪了（非UP角色），下一个SSR必是UP（类似原神）
  天井：200抽可直接兑换任意SSR
  
  → 最坏情况160抽（约23000钻石 ≈ ¥2560）必得UP角色
  → 平均期望80抽（约12800钻石 ≈ ¥1400）得到UP角色
  → 非酋保护机制确保不会无限歪
```

#### 📅 月卡 & 基金

```
小月卡（30元/月）：
  购买立即获得：300钻石
  每天登录获得：100钻石
  30天总计：300 + 100×30 = 3300钻石
  性价比：30元获得3300钻石（正常充值需约330元）→ 11倍！

大月卡（68元/月）：
  购买立即获得：680钻石
  每天登录获得：150钻石 + 50体力 + 1张酒令
  30天总计：680 + 150×30钻石 + 30酒令
  
成长基金（128元，终身一次）：
  每达到一个等级里程碑获得钻石：
  Lv.10 → 200💎
  Lv.20 → 400💎
  Lv.30 → 600💎
  Lv.40 → 1000💎
  Lv.50 → 1500💎
  Lv.60 → 2000💎
  总计：5700钻石（128元，正常充值需约600元）
  
  → 设计目的：长期留存锚点，玩家为了"回本"持续玩
```

#### 🏆 赛季通行证（Battle Pass）

```
价格：68元/赛季（45天）
免费线：30级，提供基础资源
付费线：30级，提供限定内容

付费线独占奖励：
  - 赛季限定武将皮肤 ×1
  - 赛季限定蹴鞠外观 ×1
  - 赛季限定头像框 ×1
  - SSR碎片 ×10（可选角色）
  - SR角色自选券 ×1
  - 钻石 ×2000（分布在各级）
  - 限定称号 ×1
```

#### 👘 皮肤/外观商店

```
价格区间：
  球衣皮肤（普通）：128~328钻石
  球衣皮肤（限定）：480~648钻石
  蹴鞠外观：        168~328钻石
  踢球特效：        268~488钻石
  进球庆祝动作：    198~368钻石
  场地主题：        488~888钻石
  
设计原则：
  - 皮肤不影响任何数值
  - 部分皮肤可通过游玩获得（成就/排位奖励）
  - 限定皮肤有"限时返场"机制（6个月后可能复刻）
```

## 三、玩家留存设计

### 3.1 每日循环

```
每日任务（每天刷新）：
  □ 完成3场PVE比赛        → 金币×5000
  □ 完成1场PVP比赛        → 金币×3000
  □ 使用3次武将技能        → 技能书×1
  □ 达成1次3连击以上       → 经验丹×5
  □ 完成全部每日任务       → 钻石×50 ← 重要留存锚
  
每日登录奖励（7天循环）：
  Day1: 金币×10000
  Day2: 体力×60
  Day3: R碎片×5
  Day4: 技能书×3
  Day5: SR碎片×3
  Day6: 酒令×1
  Day7: SSR碎片×1 ← 7天连续登录的强力奖励
```

### 3.2 每周循环

```
周常任务（周一刷新）：
  □ 完成20场比赛           → 钻石×100
  □ 升级任意武将3次         → 金币×20000
  □ Roguelike通关1次       → 随机秘技×1
  □ PVP获胜5场             → 段位积分×500
  □ 完成全部周常            → 酒令×3

周末限时活动：
  「梁山大比武」— 周六日开放，特殊规则PVP锦标赛
  「蹴鞠嘉年华」— 特殊Roguelike修改器（更疯狂的秘技组合）
```

### 3.3 留存关键节点

```
Day 1  留存目标 50%：新手引导确保玩家理解弹射 + 获得第一个SSR
Day 3  留存目标 35%：解锁Roguelike模式 + 完成第一章
Day 7  留存目标 25%：PVP开放 + 七日登录奖SSR碎片
Day 14 留存目标 18%：赛季通行证价值显现 + 首个SR武将升星
Day 30 留存目标 12%：成长基金回本过半 + 首个SSR升星
Day 60 留存目标 8% ：稳定PVP排位 + 第二赛季开始
```

## 四、首发运营计划

### 4.1 预热期（世界杯期间 2026年6月~7月）

```
社交媒体矩阵：
  - 抖音/TikTok：发布概念视频、角色预告
  - 微博：角色投票、互动话题
  - B站：开发日志视频、策划分享
  - 小红书：角色立绘、美术风格展示
  - Twitter/X：英文版预告，面向海外用户
  
内容策略：
  每周发布1个角色预告（先SSR角色）
  世界杯进球时发布对应水浒蹴鞠梗图（蹭热度）
  发起"你最想谁上场"投票（筛选核心用户）
  
预约目标：10万+预约
```

### 4.2 上线首周

```
首发活动：
  1. 「七日登录送SSR」— 登录7天送燕青（封面角色）
     → 保证每个玩家都有一个SSR，降低流失
  
  2. 「首充双倍」— 任意金额首充，钻石数×2
  
  3. 「梁山招贤」— 限时UP池（首期：林冲），概率提升至3%
  
  4. 「新手挑战赛」— 完成指定任务获得大量资源
  
  5. 「邀请好友」— 每邀请1人获得酒令×3
```

### 4.3 财务预估（保守）

```
假设条件：
  - 首月下载量：50,000（iOS + 自然量 + 买量）
  - 付费率：5%
  - 月ARPU：$4
  - 月活留存率：30%衰减

收入预估（美元）：
  Month 1:  50,000 × 5% × $4    = $10,000
  Month 2:  35,000 × 6% × $4.5  = $9,450
  Month 3:  25,000 × 7% × $5    = $8,750
  Month 6:  15,000 × 8% × $5.5  = $6,600
  Month 12: 10,000 × 10% × $6   = $6,000
  
  首年总收入预估：$80,000~120,000（约56~84万人民币）

成本预估：
  开发（6个月，3~5人团队）：约30~50万人民币
  美术外包：约10~20万人民币
  服务器（首年）：约3~5万人民币
  买量/营销：约5~10万人民币
  苹果税（30%）：从收入中扣
  
  → 保守情况下首年基本打平或小亏
  → 如果出圈（下载量×5），则首年可盈利
  → 关键在于长期运营和口碑传播
```

## 六、风险与应对

| 风险 | 可能性 | 影响 | 应对方案 |
|------|--------|------|---------|
| 弹射手感做不好 | 中 | 致命 | 预研阶段反复测试，参考怪物弹珠 |
| 美术成本超支 | 高 | 大 | 先做20个核心角色，N卡用简化美术 |
| PVP不平衡 | 高 | 中 | 等级压缩+禁选+赛季调整 |
| 抽卡被骂太氪 | 中 | 中 | 保底机制+免费送SSR+N卡有用 |
| 世界杯热度过后下载量暴跌 | 高 | 中 | 国潮IP有长期价值，不完全依赖世界杯 |
| 苹果审核不过 | 低 | 大 | 提前了解审核规则，避免抽卡合规问题 |
| 竞品模仿 | 中 | 中 | 先发优势+持续更新+社区运营 |

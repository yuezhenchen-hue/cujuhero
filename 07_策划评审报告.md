## 🔴 致命问题（不改必翻车）

### 1. Combo速度加成是乘法叠加，数值会爆炸

**位置：** 01_物理引擎设计.md 第437~452行

当前设计：球每碰到一个队友，`combo_speed_bonus = 1 + combo * 0.10`，然后 `ball.vel *= combo_speed_bonus`。

问题在于这是**每次碰撞都乘以一个递增系数**：

```
第1次连击：vel × 1.10
第2次连击：vel × 1.10 × 1.20
第3次连击：vel × 1.10 × 1.20 × 1.30
第4次连击：vel × 1.10 × 1.20 × 1.30 × 1.40
第5次连击：vel × 1.10 × 1.20 × 1.30 × 1.40 × 1.50
= 原始速度 × 3.6倍！
```

再叠加武将的 `kick_power` 加成（最高1.5倍），球的实际速度可达 **原始的5.4倍**。此时：
- 球速远超守门员扑救能力，进球变成必然
- 拦截判定中 `speed_penalty` 封顶0.5，面对超高速球毫无意义
- 物理引擎的碰撞检测可能因为高速穿透失效（见问题#3）

**修改建议：**
```python
# 方案A：改为固定加成（推荐）
combo_speed_bonus = 1 + min(combo, 5) * 0.08  # 封顶1.4倍
ball.vel = ball.vel.normalized() * (base_speed * combo_speed_bonus)

# 方案B：改为对数增长
combo_speed_bonus = 1 + 0.15 * log(1 + combo)  # 递减增长
```

### 2. 花荣「神箭射门」在PVP中是毁灭性的

**位置：** 02_角色系统设计.md 花荣主动技能

技能效果："100%命中球门死角，无视守门员"。冷却4回合。

在PVP中（每方约10次弹射机会），花荣可以释放 **至少2次**必进球。如果配合：
- 宋江被动"拦截判定必定失败一次"：花荣一定能碰到球
- 宋江终极"聚义厅"刷新全队冷却：花荣可以连续释放
- 柴进SR被动"全队技能冷却-1"：花荣冷却变3回合

这意味着一套"花荣+宋江+柴进"的组合可以**稳定每场进4~5球**，而且完全无法防守。

PVP将退化为："你有花荣吗？有→赢，没有→输"。

**修改建议：**
- 花荣技能改为"射门命中率极高（90%）"而非100%，给守门员一线生机
- 增加前置条件：必须 combo ≥ 2 后才能触发（不能直接射门）
- PVP中该技能额外增加冷却（+2回合），变为6回合冷却
- 或者更根本地：技能改为"大幅提高弧线精度+球速"而非"绕过物理引擎"

**设计原则：任何技能都不应该绕过核心物理系统。一旦开了这个口子，玩家体验会割裂。**

### 3. 物理引擎缺少高速穿透保护（CCD）

**位置：** 01_物理引擎设计.md 整体

当前设计中，`MAX_LAUNCH_SPEED = 1800`，`DT = 1/60`。这意味着物体一帧移动距离最高：

```
1800 × (1/60) = 30 逻辑单位/帧
```

球的半径是20，武将最小半径35。理论上不会穿透（30 < 20+35=55）。

但考虑到：
- 多重加成后球速可达 5000+（见问题#1的combo爆炸）
- 技能加速（林冲冲射 ×1.5、索超 ×1.8）
- 一帧移动距离可达 80+，完全可以穿过一个武将

**结果：球直接穿过守门员进球，或穿过后卫不触发拦截判定。**

**修改建议：**
```python
# 方案A：亚步进检测（Sub-stepping）
def update_with_substeps(self, dt):
    max_step = min(body.radius, target_radius) * 0.5  # 每步不超过半径
    distance = body.vel.length() * dt
    steps = max(1, int(distance / max_step))
    sub_dt = dt / steps
    for _ in range(steps):
        body.pos += body.vel * sub_dt
        self.check_collisions(body)

# 方案B：射线检测（Ray casting）  
# 每帧对高速物体做一次从旧位置到新位置的射线检测
# 如果射线与任何碰撞体相交，在交点处处理碰撞
```

### 4. 宋江/吴用的特殊位置导致阵容规则混乱

**位置：** 02_角色系统设计.md 宋江&吴用设计

当前规则：
- 标准阵容 = 5人上场（1GK + 4外场）
- 吴用 = "教练位"（不上场）
- 宋江 = "队长位"（上场但不参与弹射，自动移动）

**问题矩阵：**

| 情况 | 结果 | 问题 |
|------|------|------|
| 用吴用+宋江+3外场+1GK | 实际可弹射的只有3人 | 玩家操作空间被严重压缩 |
| 不用吴用不用宋江 | 4人可弹射+1GK | 正常，但两个SSR废了 |
| 宋江上场占1个位置 | 场上有6个碰撞体（含宋江），对手只有5个 | 不公平 |
| 宋江不占位置 | 那他的碰撞体在哪？能被球碰到触发combo吗？ | 规则不清 |

同时，阵容编辑界面设计中"教练：吴用 队长：宋江"暗示他们是额外位置。
但羁绊"梁山之巅"要求宋江+卢俊义+吴用同时在阵容中——如果阵容只有5+1+1=7个位置，那还能选几个战斗角色？

**修改建议：**
- 明确定义"大阵容"概念：5上场 + 1教练 + 1队长 + 2替补 = 9人阵容
- 宋江上场时**替代1个外场位置**，但获得"光环效果"补偿操作减少
- 或者更简洁：宋江重新设计为可弹射的MF角色（弱弹射但有强buff），取消"不能弹射"的设定
- 吴用保持教练位不上场，但明确教练位是额外的第6个阵容位

## 🟠 严重问题（不改会很痛）

### 5. 守门员AI不考虑弧线球，TEC属性将统治游戏

**位置：** 01_物理引擎设计.md 守门员AI predict函数

```python
predicted_x = ball.pos.x + ball.vel.x * time_to_goal
```

这是纯线性预测。但游戏的核心卖点之一是**弧线球**（马格努斯效应）。

结果：任何带旋转的球（TEC > 50 的武将踢出的球都有spin），守门员的预判必定是错的。弧线球将成为无脑射门方式，破坏了弹射策略的深度。

**修改建议：**
```python
def predict_ball_position(self, ball):
    """预判需要考虑旋转偏移"""
    if abs(ball.vel.y) < 1:
        return ball.pos.copy()
    
    time_to_goal = abs((self.goal_y - ball.pos.y) / ball.vel.y)
    
    # 考虑马格努斯效应的累积偏移
    spin_offset_x = 0
    if ball.spin.length() > 0.01:
        spin_offset_x = ball.spin.x * MAGNUS_COEFFICIENT * time_to_goal * time_to_goal * 0.5
    
    # 预判精度受 GK 属性影响（低属性GK仍然会被骗）
    accuracy = self.body.hero_data.goalkeeping / 100.0
    predicted_x = ball.pos.x + ball.vel.x * time_to_goal + spin_offset_x * accuracy
    
    return Vector2(predicted_x, self.goal_y)
```

这样高GK属性的守门员能读懂弧线，低GK属性的读不懂——产生有意义的属性差异。

### 6. PVP中大量随机机制会劝退竞技玩家

**涉及位置：** 多处

| 随机机制 | 位置 | 问题 |
|---------|------|------|
| 武松醉拳随机偏转15~30度 | 角色技能 | 释放者自己也不知道球往哪飞 |
| 拦截判定 `random.random() < final_chance` | 物理碰撞事件 | 同一局面，两次结果可能完全不同 |
| 吴用被动 30%概率预知对手方向 | 角色技能 | PVP胜负看脸 |
| 白日鼠白胜 30%概率复制SSR技能 | N卡被动 | 极端情况下N卡=SSR |
| 公孙胜每3回合随机风向 | 角色技能 | 不可控环境因素 |

弹射游戏的核心乐趣是**"我算准了角度和力度"**。如果结果被随机干扰，玩家会觉得"不是我菜，是运气差"——这在PVP中是致命的负面体验。

**修改建议：**
- PVP模式中，所有"概率判定"改为"确定性判定"
  - 拦截：DEF > 球速阈值 → 必定拦截；否则必定失败
  - 武松醉拳：改为"固定偏转模式"（如固定30度→-30度交替），玩家可以预判
  - 吴用被动：PVP中改为"每3回合必定触发一次"而非30%每回合
- PVE中保留随机性没问题（增加趣味和复玩性）

### 7. 108角色美术资产量级被严重低估

**位置：** 02_角色系统设计.md 美术资产列表 + 05_技术选型.md 时间线

按文档列出的每个角色需要的资产：
- 1张卡面立绘（1024×1536）
- 1张头像
- 6个Spine动画（idle/kick/hit/skill/celebrate/slingshot）
- 4张对话差分
- 2套技能特效
- 至少1套皮肤

**单个SSR角色的美术工作量估算：**
- 立绘：3~5天（一个优秀原画师）
- Spine骨骼+6个动画：5~7天（一个Spine动画师）
- 对话差分：1~2天
- 技能特效：2~3天
- 合计：**约11~17天/角色**

12个SSR × 15天 = **180天**（约6个月，一个美术全职干）
再加24个SR（简化×0.7倍工作量）= 252天
再加36个R（简化×0.4倍）= 216天
再加36个N（简化×0.2倍）= 108天

**总计：756天美术工作量 ≈ 3个美术师做1年**

但时间线中Phase 2~3只有"1~2美术 + 1动画"用12周完成这些。这是完全不可能的。

**修改建议：**
- MVP砍到25~30个角色（5SSR + 8SR + 12R + 5N）
- N卡不做Spine动画，使用静态图+简单位移代替
- R卡共享部分动画骨骼（同类型角色复用skeleton）
- 分批上线：首发30角色，每月更新5~8个
- 或大幅增加美术预算（外包给专业2D游戏美术工作室）

是的，这意味着**赶不上2026世界杯上线**。但一款烂游戏赶上世界杯也没用，一款好游戏晚3个月不会死。

### 12. 抽卡缺少"软保底"机制

当前保底：硬保底80抽。

参考原神的设计：从第74抽开始，SSR概率从2%线性提升到100%（第90抽）。这个"软保底"让玩家在60~80抽之间获得惊喜感。

当前设计下，玩家前79抽每次只有2%概率，第80抽突然100%——体验上缺少"越抽越接近"的期待感。

**修改建议：**
```
第1~60抽：SSR概率 = 2.0%
第61抽起：SSR概率 = 2.0% + (当前抽数 - 60) × 3.0%
第70抽：SSR概率 = 32%
第75抽：SSR概率 = 47%
第80抽：SSR概率 = 62%（仍非100%）
硬保底：第90抽 = 100%
```

这样既延长了保底（90比80多消费），又让玩家从第60抽开始感到"快了快了"的兴奋感。

### 13. 新手引导（FTUE）设计严重不足

文档中新手引导只有一句"序章教学关3场"。但对于一款核心操作是弹射的游戏，FTUE极其关键。

需要细化的内容：
- 第1次弹射：强制引导拉弹弓方向，有巨大箭头提示
- 第1次进球：触发极其华丽的慢动作+庆祝，给予强烈正反馈
- 第1次combo：专门设计一个2连击必然发生的关卡（固定站位）
- 第1次技能：用燕青的弧线球技能，因为视觉效果最惊艳
- 第1次失败：给予安慰机制（"再来一次？"），不让玩家卡关
- 教学结束奖励：免费获得一个SSR（燕青），让玩家感到满足

Day 1留存50%的目标，完全取决于前5分钟的体验。

### 14. "国潮水墨 × 热血动漫"美术风格有冲突风险

水墨画强调留白、意境、柔和笔触。
热血动漫强调夸张、速度线、高饱和色彩。

两者在美术理论上是矛盾的。做好了是"惊艳"（参考《大圣归来》），做不好是"四不像"。

- 需要在立项阶段就产出3~5张**概念图**，验证风格融合度

- 建议参考《阴阳师》而非《大闹天宫》，前者更接近"水墨+角色魅力"的成功案例
- 场景用水墨，角色用偏二次元的精致画法——这是被验证过的折中方案

### 17. PVP"3分钟+各10次弹射"时间可能不够

每次弹射：
- 决策时间 3~5秒
- 弹射动画 0.5秒
- 物理模拟 2~4秒
- 回合结算 1秒
- 合计：6.5~10.5秒/回合

20回合（双方各10次）× 8秒 = 160秒 ≈ 2.7分钟

看似刚好，但没算：
- 技能释放动画（2~3秒）
- 进球庆祝（3秒）
- 教练技能决策时间

实际一场可能需要4~5分钟。

但与1.4节"SSR属性总和范围380~450"矛盾——燕青363不在SSR范围内。

需要统一：要么调高燕青属性，要么把SSR范围下限调到330。

## 补充建议（新增内容）

### A. 建议增加"快速模式"作为核心玩法的补充

当前每场比赛需要3~5分钟。对于手游来说偏长。建议增加：
- **30秒闪电赛**：各弹射3次，进球多者胜
- 适合通勤、排队等碎片场景
- 可以作为每日任务的快速完成方式

### B. 建议增加"重播/分享"功能

弹射类游戏天然适合产出精彩瞬间（神角度进球、5连击穿透射门等）。
- 自动录制每场比赛的精彩瞬间
- 一键分享到社交媒体（带游戏水印）
- 这是最廉价也最有效的传播方式

### C. 建议增加"蹴鞠百科"模块

既然用了水浒传IP，应该最大化利用文化价值：
- 每个角色有"水浒原著故事"标签页
- 解锁角色时播放该角色的水浒典故
- 蹴鞠历史科普（蹴鞠的起源、发展、与FIFA的关系）
- 这不仅有教育意义，也能让App Store编辑推荐的概率大幅提高（文化类游戏优先推荐）

### D. 建议"高俅"不要做解锁角色，改为赛季限定

高俅作为最终BOSS，是整个剧情的"反派"。让玩家轻松解锁使用，会削弱他的叙事价值。

建议：
- 通关后获得高俅碎片（不足以直接解锁）
- 首个赛季"宿敌降临"UP池中高俅为限定角色
- 这样既保持了叙事完整性，又创造了付费点

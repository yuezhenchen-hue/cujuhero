## 项目一句话

用水浒108将组队，以弹射玩法踢蹴鞠，闯荡江湖赛场的Roguelike竞技手游。

## 核心卖点

- **独一无二的题材**：水浒传 × 蹴鞠，市面零竞品
- **世界杯借势**：2026世界杯热度加持
- **零版权成本**：水浒传公共领域IP
- **验证过的玩法模型**：弹射（怪物弹珠验证）+ Roguelike（杀戮尖塔验证）
- **国潮文化出海**：水墨风格对海外有强烈吸引力
